## Краткое описание приложения

Приложение представляет из себя репозиторий информации о спортивной школе с гибкой интеграцией на стороне спортсменов для обновления информации.

## Программное решение
Сервисы:  
- Сервер авторизации;
- API;
- PostgreSQL БД.
